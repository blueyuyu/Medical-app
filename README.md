## 医疗小程序 uni-app 前端

这是一个基于vue3+vite 的uni-app 医疗小程序的前端


## 运行

请使用 Hbuilder X 

```
npm i
```

下载完node_modules包之后，查看uni_modules是否完整，
以及下载完成 uni_modules 之后，即可将通过上方运行（运行到浏览器）启动项目。

## 运行环境

node 版本要求 > 8.9
